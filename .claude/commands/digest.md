整理飞书群聊的最近消息。

## 参数
- $ARGUMENTS：群名关键词，可选指定天数（默认 3 天）

## 执行步骤
1. 用 `search_contacts` 搜索匹配的群聊
2. 用 `read_messages` 读取最近 N 天的消息（计算 start_time 时间戳）
3. 按以下标准筛选有价值内容：
   - 包含可操作的具体方法论或工具推荐 → 高价值
   - 包含带数据支撑的趋势分析 → 高价值
   - 包含可复现的实践经验 → 高价值
   - 纯寒暄/水聊 → 跳过
4. 向用户汇报：
   - 共读取 X 条消息
   - 筛选出 Y 条有价值内容
   - 涉及的主题分类
5. 等用户确认后，用自己的语言概括核心观点

## 注意
- 读消息使用官方 API（需配置 LARK_APP_ID）
- 如有外部链接可用 playwright 抓取全文
